This file documents the Electron icon usage for the package.

The app icon is referenced at `../docs/img/favicon.ico` in `electron/package.json`.

If a dedicated `electron/resources` icon set is desired, add:
- `electron/resources/icon.ico` for Windows
- `electron/resources/icon.icns` for macOS

Then update `electron/package.json` to point to the new icon path.
